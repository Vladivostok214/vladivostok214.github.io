# Radio Caleuche Online 🌊📻

**La nueva plataforma de comunicación insular surcando los mares digitales de Chilwé.**

Radio Caleuche es un proyecto de vanguardia dedicado a potenciar la cultura y la actualidad de Chiloé mediante una experiencia digital inmersiva y de alto impacto.

## ✨ Características Principales

- **Diseño Vanguardista:** Estética suiza minimalista en Dark Mode profundo.
- **Reproductor Central:** Interfaz de audio customizada con visualizador de frecuencias e indicador "ON AIR".
- **Optimización Total:** Rendimiento ultra-rápido, SEO avanzado y soporte para Open Graph.
- **Full Responsive:** Experiencia de usuario consistente desde móviles hasta monitores 4K.

## 🚀 Tecnologías

- **Diseño:** Swiss Minimalist Style / Montserrat Typography.
- **Core:** HTML5 / Modern CSS (Variables & Fluid Design).
- **Interacción:** jQuery / Custom Audio Logic.
- **Despliegue:** Optimizado para Vercel / GitHub Pages.

## 🛠️ Desarrollo

El sitio es estático y no requiere pasos de construcción (Build steps).

1. Clona el repositorio.
2. Abre `index.html` en tu navegador.
3. Para una mejor experiencia, usa un servidor local:
   ```bash
   npx serve .
   ```

## 📐 Estructura

```text
├── index.html       # Punto de entrada optimizado
├── css/style.css    # Estilos de vanguardia
├── js/main.js       # Lógica de interfaz y reproductor
├── img/             # Activos visuales optimizados

```
